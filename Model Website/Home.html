Description of the Project


MRI Scans can often be difficult to read, and given the severity of diseases like brain cancer, developing automated classification techniques which have higher accuracy than manual classification is incredibly helpful for developing good diagnoses.

In this project, I used a Kaggle dataset of ~2,000 MRI scans of various tumor types to develop a machine learning model to create brain tumor diagnoses.

I used Tensorflow's framework and pretrained image classification model: EfficientNetB0 to develop a transfer learning model that identifies tumor types with an accuracy of ~85%.
